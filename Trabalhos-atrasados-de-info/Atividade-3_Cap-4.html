Seção 2 — Ranking de Tecnologias

  1. HTML

    Base estrutural de qualquer página web e ponto de partida para o desenvolvimento frontend.

  2. CSS

    Responsável pela aparência visual e pela experiência do usuário.

  3. JavaScript

    Permite criar interatividade e funcionalidades dinâmicas nas aplicações web.

  4. Banco de Dados

    Fundamental para armazenar, organizar e recuperar informações.

  5. APIs

    Essenciais para a comunicação entre sistemas e integração de serviços modernos.

Seção 3 — Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para estruturar páginas web.
CSS
Linguagem utilizada para estilizar elementos HTML.
JavaScript
Linguagem de programação voltada para interatividade em páginas web.
Também pode ser utilizada no backend através do Node.js.
DOM
Modelo de objetos que representa a estrutura de um documento HTML.
API
Conjunto de regras que permite a comunicação entre sistemas.
Pode fornecer acesso a dados, autenticação ou serviços externos.
HTTP
HyperText Transfer Protocol
Protocolo utilizado para transferência de informações na Web.
URL
Uniform Resource Locator
Endereço utilizado para localizar recursos na Internet.
DNS
Sistema responsável por converter nomes de domínio em endereços IP.
